# Dimori

To install library needed
```sh
    yarn
```

To deploy smart contract to ganache net work
```sh
    cd hardhat
    npx hardhat run scripts/deploy-dimori.js --network ganache
```

Test smart contract by using hardhat test
```sh
    npx hardhat test
```

Run app 
```sh
    yarn start
```
